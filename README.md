# 💕 Love Proposal Website

A beautiful, romantic multi-page website created for a special love proposal. Built with React, Vite, and TailwindCSS.

## Features

- **Home Page**: Stunning hero section with romantic messaging and call-to-action
- **Our Story Page**: Interactive timeline showcasing the journey of love
- **Memories Page**: Beautiful grid of precious memories and special moments
- **Proposal Page**: Heartfelt love letter with interactive response system

## Tech Stack

- **React 18** - Modern UI library
- **Vite** - Fast build tool and dev server
- **React Router** - Client-side routing
- **TailwindCSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icon library
- **Framer Motion** - Smooth animations

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open your browser to the URL shown in the terminal (typically `http://localhost:5173`)

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Customization

### Personalize the Content

Each page contains customizable content. Edit the following files to make it your own:

- `src/pages/Home.jsx` - Landing page content
- `src/pages/OurStory.jsx` - Your love story timeline
- `src/pages/Memories.jsx` - Special memories and moments
- `src/pages/Proposal.jsx` - The proposal message

### Change Colors

The color scheme can be customized in `tailwind.config.js`. The current theme uses rose and pink gradients for a romantic feel.

### Add Your Own Images

Replace the placeholder content with your own photos and memories to make it truly personal.
